## 🚀 Overview

**Lumina Books** is a powerful desktop application designed specifically for bookstores.  
It combines **inventory management, POS billing, and analytics** into one seamless system — all running **100% offline**.

✔ No subscriptions  
✔ No cloud dependency  
✔ Full data ownership  

---

## ✨ Key Features

### 📦 Inventory Tracking
- Auto low-stock alerts  
- Bulk import support  
- Category & supplier management  
- Fast search across titles  

---

### 🧾 POS System
- Lightning-fast billing  
- Barcode scanner support  
- Instant receipt generation  
- Discount & pricing rules  

---

### 📊 Sales Analytics
- Daily & weekly reports  
- Top-selling books  
- Profit insights  
- Clean visual dashboards

### ⚡ Offline First
